# Practica con React

Un proyecto sencillo de práctica utilizando **React 19** y **htm** para ejecutar componentes directamente en el navegador sin necesidad de compilación previa.

## Características
- **React 19**: Versión más reciente de React.
- **Sin compilación**: Utiliza `htm` para manejar JSX de forma nativa en el navegador.
- **Diseño Premium**: Estilos modernos con gradientes y modo oscuro.
- **Servidor Simple**: Se puede ejecutar con cualquier servidor estático (ej. `python -m http.server`).

## Cómo ejecutarlo
1. Clona este repositorio.
2. Abre un terminal en la carpeta del proyecto.
3. Ejecuta un servidor local:
   ```bash
   python -m http.server 8000
   ```
4. Abre `http://localhost:8000/app.html` en tu navegador.
